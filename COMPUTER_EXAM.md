# 机考 讲解

0.如果大一上过C语言相关课程，期末使用一个蓝色背景考试系统机考，那你应该不用往下看了，机考也是这个系统

1.题型在md已经说了，5*8分

2.使用Dev C++软件，在注释规定的区域内写代码（自己可以在外面写函数，然后在注释区域内调用）（注意：超出注释规定区域写的代码不会被判分识别到），随时可以运行代码，确保正确符合题
意后再保存（Ctrl+S）后，再回到系统进入下一题，在交卷之前可以随时回到某道题，修改已保存的代码

3.给分要求是：结果正确则满分，不限时间、空间复杂度，若结果出错，则会按写的代码中按点给分（题目应该难不到你们，尽量不要走到这一步，这个系统给分机制可能比较死板）

4.难度以考察C语言基础能力为主，偏易，基本都以排序、数组、条件、循环、数学运算、结构体为主，去年未考文件读取、指针等内容，也不会考复杂算法

5.还是md说过的，会写一种排序算法，到时候会用


6.不用过于担心机考，题目以考察C基础为主，不会像力扣算法之类的难度的

7.Dev C++带的C标准比较旧，不支持for(int i = 0;i < x;i++)写法，需要先int i = 0;for(i = 0;i < x;i++)，否则会报错

8.（计算机学院不公布任何环节的分数🥲，其实机考写完了就会自动判出分了）
